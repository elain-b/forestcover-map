# mapbox-app ver.3

### 作ったもの
- Google Earth Engineよりgeojsonファイルで森林量をダウンロード
- Mapboxを地図の土台に森林量の可視化

### 環境
- HTML

### うまくいった
- jsファイルに書いていたコードをhtmlのscriptタグの中に書いた。
- linkタグとscriptタグでmapboxのcssとjsを読み込んだ。
